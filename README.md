# C Header Generator (makeh)

Автоматическая утилита на Python для генерации профессиональных заголовочных файлов (`.h`) на основе файлов реализации (`.c`). 

## Особенности
- **Интеллектуальный парсинг**: Утилита использует автомат состояний для корректной обработки функций, структур и перечислений.
- **Сохранение лицензий**: Автоматически переносит первый блок комментария (лицензионную шапку) из `.c` в `.h`.
- **Поддержка C++**: Генерирует блоки `extern "C"` для совместимости с C++.
- **Чистый интерфейс**: Игнорирует внутренние функции (`static`), `main` и управляющие конструкции (`if`, `while`, `for`).
- **Обработка препроцессора**: Корректно переносит макросы и директивы инклудов.

## Использование

```bash
python makeh.py путь/к/вашему_файлу.c
